## About QuitTxT App

QuitTxT App is a Flutter-based mobile application that provides a Rich Communication Services (RCS) messaging experience enhanced with AI features. It aims to deliver a modern chat interface supporting various media types and intelligent interactions to help users with smoking cessation.

## Key Features

- RCS Messaging with rich media support
- Gemini AI integration for intelligent responses
- Support for images, GIFs, videos, and YouTube previews
- Quick replies and suggestions
- Multi-language support
- Thread replies and reactions
- Firebase backend integration
