## Feature Overview

The QuitTxT App offers a comprehensive set of features designed to create an engaging messaging experience with AI-powered support for smoking cessation:

### RCS Messaging

A modern chat interface with advanced messaging capabilities:
- Text messaging
- Read receipts and typing indicators
- Thread replies
- Message reactions
- Conversation management

[Learn more about RCS Messaging](rcs-messaging.md)

### Gemini AI Integration

Integration with Google's Gemini AI for intelligent interactions:
- AI-powered responses
- Context-aware suggestions
- Enhanced quick replies with AI generation
- Natural language understanding

[Learn more about Gemini AI Integration](gemini-integration.md)

### Media Handling

Support for rich media in conversations:
- Image sharing (camera and gallery)
- GIF support (both local and online sources)
- Video sharing and playback
- YouTube link previews and embedded players
- File sharing and document preview

[Learn more about Media Handling](media-handling.md)

### Quick Replies

Interactive reply buttons for faster communication:
- User-generated quick reply options
- Keyword-based automatic suggestions
- Gemini-generated contextual suggestions
- Visual distinction between different types of quick replies

[Learn more about Quick Replies](quick-replies.md)

## Additional Features

### Multi-Language Support

The app supports multiple languages through a localization system:
- English and Spanish language support
- Dynamically switchable in settings
- Uses Flutter's localization framework

### Platform Adaptation

The app provides platform-specific UI implementations:
- iOS (Cupertino) design for iOS devices
- Material Design for Android devices
- Adaptive behavior based on platform detection

### Environment Switching

Support for different environments:
- Development
- Production
- Easy switching between environments

For more detailed information on each feature, please refer to the individual documentation pages.
